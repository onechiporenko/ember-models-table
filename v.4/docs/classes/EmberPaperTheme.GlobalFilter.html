Class GlobalFilter

Global filter element used within ModelsTable.

Its value is used for all data-items and for each columns filterField.

Usage example:

<ModelsTable @data={{this.data}} @columns={{this.columns}} as |MT|>
  <MT.GlobalFilter />
  {{! .... }}
</ModelsTable>

Block usage example:

<ModelsTable @data={{this.data}} @columns={{this.columns}} as |MT|>
  <MT.GlobalFilter>
    {{input value=MT.globalFilter}}
    <button disabled={{if MT.globalFilterUsed "disabled"}} {{on "click" this.clearGlobalFilter}}>
      Clear Global Filter
    </button>
  </MT.GlobalFilter>
  {{! .... }}
</ModelsTable>
